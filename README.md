## SDD (Source of Truth)
La fuente de verdad para este proyecto está en /specs:
- /specs/spec.md (requisitos + criterios de aceptación)
- /specs/ui.md (wireframes + reglas UI)
- /specs/plan.md (fases)
- /specs/tasks.md (tareas atómicas)
- /specs/data-schema.json (workspace portable de ejemplo)

Reglas:
- No implementes nada que no esté en spec.md.
- Implementa 1 tarea cada vez (tasks.md).
- Después de cada tarea: actualiza tests y valida criterios relacionados.
